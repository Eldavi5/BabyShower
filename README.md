# NewProyectMona

Invitación web para baby shower con estética baby bloom, tonos azul y blanco, animaciones suaves, fondo decorativo con ositos, música ambiental y espacios preparados para fotos del bebé.

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS v4
- App Router

## Desarrollo
```bash
npm install
npm run dev
```

## Producción
```bash
npm run build
npm run start
```
